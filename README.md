# 🏥 Healthcare Dashboard – AI Analysis (Excel Project)

## 📌 Overview

This project is an interactive **Healthcare Dashboard built in Microsoft Excel** to analyze patient data, medical conditions, and healthcare costs. The dashboard provides insights into relationships between factors such as smoking, diabetes, heart issues, weight status, and medical charges.

---

## 🎯 Objectives

* Analyze healthcare costs based on patient conditions
* Identify patterns between lifestyle factors (smoking, weight) and diseases
* Explore relationships between diabetes, heart issues, and age
* Provide an interactive dashboard for easy data exploration

---

## 🛠️ Tools & Techniques Used

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Preparation
* Slicers for interactivity
* Conditional Formatting
* Excel formulas (SUM, COUNT, IF, etc.)

---

## 📊 Dashboard Features

* 🔄 Interactive slicers:

  * Weight Status
  * Heart Issues
  * Diabetes Status

* 📈 Visualizations include:

  * **Cancer vs Smoker Analysis** (Donut Chart)
  * **Charges by Weight Status**
  * **Charges by HbA1c (Diabetes Indicator)**
  * **Age vs Diabetes Analysis**
  * **Charges by Heart Issues**
  * **City Tier vs Population/Charges**

---

## 📈 Key Insights

* Smoking is strongly associated with higher cancer cases
* Overweight individuals tend to have higher medical charges
* Diabetes (HbA1c levels) impacts healthcare costs significantly
* Age plays a role in diabetes prevalence
* Higher city tiers show increased healthcare spending
